# 프로젝트 수정 사항 (2025-12-26)

## 🎯 주요 수정 내용

### 1. 자관리자 페이지 개선
- **사이드바 슬라이드 애니메이션 수정** (`AdminPage.vue`)
  - 사이드바 토글 시 페이지 전체가 늘어나는 문제 해결
  - 메인 컨텐츠 영역을 별도 div로 감싸서 레이아웃 안정화
  - 모바일/데스크톱에서 자연스러운 슬라이드 효과 구현

### 2. 헤더 네비게이션 개선
- **관리자/회원 모드 전환 버튼 추가** (`Header.vue`)
  - ADMIN 권한 사용자에게 관리자 모드 버튼 표시
  - 관리자 페이지에서는 "회원 모드" 버튼으로 자동 전환
  - 마이페이지/관리자페이지 진입 시 카테고리 선택 자동 해제
  - 카테고리 네비게이션을 마이페이지/관리자페이지에서 숨김 처리

### 3. 지도 마커 개선
- **VenueMarker 컴포넌트 업데이트** (`VenueMarker.vue`)
  - 펄스 효과 크기를 마커 크기의 약 1.2배로 조정 (r: 12→38)
  - 펄스 효과 투명도를 더 진하게 조정 (opacity: 0.36→0.45)
  - SVG viewBox와 실제 크기 일치 (width/height: 32/38px)
  - 공연 개수 뱃지 위치 및 스타일 개선

- **MapModal 마커 상호작용 개선** (`MapModal.vue`)
  - 공연장명 라벨 위치를 더 위로 조정 (yAnchor: 2.5→3.5)
  - 마커 호버 시 라벨과 마커가 겹치지 않도록 수정
  - 호버 라벨이 마커를 가려 깜빡이는 현상 해결
  - 공연장 클릭 시 즉시 지도 이동하지 않고, 길찾기 버튼 클릭 시에만 지도 중심 이동
  - 줌 레벨에 따른 라벨 표시/숨김 기능 유지

### 4. UI/UX 개선
- **가격 표시 개선** (`TicketDetailPage.vue`)
  - 가격 내 천단위 쉼표는 유지하면서 "원," 뒤에만 줄바꿈
  - `word-break: keep-all` 속성 추가로 숫자 쉼표에서 줄바꿈 방지
  - 가격 정보 가독성 향상

- **관리자 페이지 스타일 수정** (`AdminVenues.vue`)
  - `@apply` 지시문을 일반 CSS로 변경하여 경고 제거
  - 코드 안정성 향상

### 5. 마이페이지 기능
- **예약 상세 정보 모달** (`MyPage.vue`)
  - 예약 번호, 공연 정보, 예매 정보, 결제 정보를 명확하게 분리
  - 공연 상세보기 버튼 추가로 원활한 네비게이션 제공
  - 예약 취소 기능 강화
